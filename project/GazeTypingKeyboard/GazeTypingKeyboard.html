Gaze-typing Keyboard

An on-screen keyboard uses new input modality,
enables new use cases,
and creates new design space. 

I'm conducting this project as my master thesis at Tobii Dynavox for people with motor impairments

I started with user studies on typing performance of several keyboards in lab experiment manner and analysed data using matplotlib in Python, then designed and prototyped the keyboard as a C#/.NET WPF application. Currently I'm refining the keyboard proto and writing academic thesis report

After agreeing with the company, I would post more project info here, sorry for blank space in storytelling of this page!

Gaze-typing is drastically different with hands-typing, as our eye is an organ used for observation, gaze-typing keyboard needs careful interaction design to use eyes as input control; eye-tracker can't offer perfect accuracy and precision, the keyboard should be error-tolerant for daily use; new gaze-typing keyboard should also offer certain familiarity to novices to lower learning cost.
